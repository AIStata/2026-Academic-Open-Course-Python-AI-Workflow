2  环境搭建

本章将介绍如何搭建完整的 Python 数据科学环境，包括 Anaconda 安装、VS Code 配置以及 Jupyter 集成。

2.1 Anaconda 生态系统配置

2.1.1 为什么选择 Anaconda？

Anaconda 是一个流行的 Python 发行版，专门为数据科学和机器学习设计。它包含了：

  • Conda 包管理器
  • 预安装的数据科学库（NumPy、Pandas、Matplotlib 等）
  • 环境隔离功能
  • Jupyter Notebook/Lab

2.1.2 安装 Anaconda 或 Miniconda

选择建议
  • Anaconda：适合初学者，包含所有常用包（约 3GB）
  • Miniconda：适合高级用户，仅包含基础系统，按需安装包（约 400MB）

2.1.2.1 Windows 安装步骤

  1. 访问 Anaconda 官网 下载 Windows 安装程序
  2. 双击运行安装程序
  3. 选择”Just Me”（仅当前用户）
  4. 选择安装路径（建议默认）
  5. 重要：勾选”Add Anaconda to my PATH environment variable”（可选，视情况而定）
  6. 点击 Install 完成安装

2.1.2.2 macOS 安装步骤

# 使用 Homebrew 安装 Miniconda
brew install --cask miniconda

# 或者从官网下载 pkg 安装包

2.1.2.3 Linux 安装步骤

# 下载 Miniconda
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

# 运行安装脚本
bash Miniconda3-latest-Linux-x86_64.sh

2.1.3 Conda 基础命令

# 查看 conda 版本
conda --version

# 更新 conda
conda update conda

# 创建新环境
conda create -n myenv python=3.11

# 激活环境
conda activate myenv          # Windows/Linux
source activate myenv         # macOS (旧版)

# 退出环境
conda deactivate

# 列出所有环境
conda env list

# 安装包
conda install numpy pandas

# 从特定频道安装
conda install -c conda-forge jupyterlab

# 导出环境配置
conda env export > environment.yml

# 从配置文件创建环境
conda env create -f environment.yml

# 删除环境
conda env remove -n myenv

2.1.4 配置国内镜像源

由于网络原因，国内用户建议配置清华或中科大镜像源：

# 配置 conda 镜像
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge
conda config --set show_channel_urls yes

# 配置 pip 镜像
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

2.2 VS Code 与 Python 深度集成

2.2.1 安装 VS Code

访问 VS Code 官网 下载对应操作系统的安装包。

2.2.2 必需扩展安装

打开 VS Code，点击左侧扩展图标（或按 Ctrl+Shift+X），搜索并安装以下扩展：

扩展名称 扩展 ID 用途
Python ms-python.python Python 语言支持
Pylance ms-python.vscode-pylance 智能代码补全
Jupyter ms-toolsai.jupyter Jupyter Notebook 支持
Jupyter Keymap ms-toolsai.jupyter-keymap Jupyter 快捷键
Quarto quarto.quarto Quarto 文档支持
YAML redhat.vscode-yaml YAML 文件支持

2.2.3 Python 扩展核心配置

打开设置（Ctrl+,），搜索以下配置项：

{
  // Python 解释器路径（自动检测或手动选择）
  "python.defaultInterpreterPath": "path/to/python",

  // 代码格式化工具
  "python.formatting.provider": "black",

  // 代码检查工具
  "python.linting.enabled": true,
  "python.linting.pylintEnabled": true,

  // 测试框架
  "python.testing.pytestEnabled": true,

  // 终端自动激活环境
  "python.terminal.activateEnvironment": true
}

2.2.4 解释器选择

  1. 打开命令面板（Ctrl+Shift+P）
  2. 输入 “Python: Select Interpreter”
  3. 从列表中选择你创建的 conda 环境

2.2.5 工作区设置

为项目创建 .vscode/settings.json：

{
  "python.defaultInterpreterPath": "${workspaceFolder}/.venv/bin/python",
  "python.analysis.extraPaths": [
    "${workspaceFolder}/src"
  ],
  "jupyter.jupyterServerType": "local",
  "files.exclude": {
    "**/__pycache__": true,
    "**/*.pyc": true,
    ".pytest_cache": true
  }
}

2.3 Jupyter 与 VS Code 整合

2.3.1 Jupyter 扩展安装与配置

VS Code 的 Jupyter 扩展（ms-toolsai.jupyter）提供了完整的 Notebook 支持，包括：

  • 交互式 Notebook 编辑
  • 变量查看器
  • 数据查看器
  • 调试支持

2.3.2 Notebook 服务器连接方式

2.3.2.1 本地服务器

VS Code 会自动启动本地 Jupyter 服务器，无需额外配置。

2.3.2.2 远程服务器

连接到远程 Jupyter 服务器：

  1. 打开命令面板（Ctrl+Shift+P）
  2. 输入 “Jupyter: Specify Jupyter Server for Connections”
  3. 选择 “Existing”
  4. 输入服务器 URL，例如：http://localhost:8888/?token=xxx

2.3.3 内核管理

2.3.3.1 查看已安装内核

jupyter kernelspec list

2.3.3.2 安装 Python 内核

# 激活目标环境
conda activate myenv

# 安装 ipykernel
conda install ipykernel

# 将环境注册为 Jupyter 内核
python -m ipykernel install --user --name myenv --display-name "Python (myenv)"

2.3.3.3 安装 Stata 内核（可选）

Stata 集成

通过 stata_kernel 可以在 Jupyter 中运行 Stata 代码。

# 安装 stata_kernel
pip install stata_kernel

# 配置内核
python -m stata_kernel.install

2.3.4 多语言支持

Jupyter 支持多种编程语言内核：

语言 内核 安装命令
Python ipykernel conda install ipykernel
R IRkernel 参见 IRkernel
Julia IJulia 参见 IJulia
Stata stata_kernel pip install stata_kernel
